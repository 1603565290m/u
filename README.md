## u - 短网址平台
短短短！短得不能再短。缩短你的超链接<br>
以打造一个高性能，可快速部署的短网址服务平台

## 系统说明：
- 基于 Spring Boot Redis实现的一个短网址平台
### 目前
- 支持web方式和api方式创建短网址

- 支持更丰富的api操作
